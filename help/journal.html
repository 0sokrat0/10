Журнал измерений

Журнал измерений предназначен для регистрации технологических параметров реакторной установки.

Ввод измерения

  1. В меню слева выбрать раздел «Журнал измерений».
  2. Нажать кнопку «Добавить запись».
  3. Заполнить поля: мощность (МВт), температура (°C), давление (МПа), нейтронный поток.
  4. Нажать «Сохранить».

Поиск и фильтрация

Используйте фильтры по дате, параметру и оператору для поиска нужных записей.

Электронная подпись

Для подписания записи нажмите «Подписать» и введите PIN-код. Подписанную запись изменить невозможно.
