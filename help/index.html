О программе

Система «Учёт параметров реакторных установок» предназначена для автоматизации ведения электронного журнала измерений, контроля технологического регламента, учёта доз облучения персонала и формирования отчётности.

Основные возможности

  • Регистрация измерений параметров реакторной установки
  • Тревожная сигнализация при нарушении регламента
  • Расчёт накопленного выгорания ядерного топлива
  • Учёт индивидуальных доз облучения персонала
  • Удостоверение записей электронной подписью
  • Формирование отчётов в форматах PDF и XLSX
