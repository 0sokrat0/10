Установка и запуск

Системные требования

  • ОС: Windows 10 (64-бит) или выше
  • Оперативная память: 4 ГБ и более
  • Свободное место: 500 МБ для установки
  • Дисплей: разрешение не менее 1920x1080
  • Среда выполнения: Python 3.10+ или .NET 6+

Установка

  1. Запустить setup.exe от имени администратора.
  2. Следовать инструкциям мастера установки.
  3. По завершении нажать «Готово».

Запуск

Двойной щелчок по ярлыку «Система учёта реакторных установок» на рабочем столе или запуск через меню «Пуск».

Удаление

Панель управления → Программы → «ReactorMonitor» → Удалить.
